# 🎙️ Discord Clone — Полный мультиюзерный клон с поддержкой Голосовых Звонков

Это высокоточный клон интерфейса и функциональности **Discord**, построенный с использованием гибкого полнотекстового стека **React + Vite** на фронтенде и **Express.js (Node.js)** на бэкенде.

Приложение полностью готово к развертыванию, поддерживает совместную синхронизацию между вкладками и разными пользователями в реальном времени, а также предоставляет доступ ко всем функциям Discord Nitro совершенно бесплатно!

---

## ✨ Основные возможности клона

1. **Мультиплеерная Синхронизация (Real-time)**:
   - Вся база данных серверов, каналов, текстовых сообщений и статусов пользователей синхронизируется в реальном времени с помощью **SSE (Server-Sent Events)** без задержек.
   - Любое количество пользователей может зайти по одной ссылке, зарегистрировать свои никнеймы и начать общаться!

2. **Создание Серверов и Каналов**:
   - Полнофункциональная левая панель навигации по серверам (с фирменными круглыми иконками и закруглениями при наведении).
   - Возможность моментально создать свой сервер, выбрать для него уникальную иконку-эмодзи. Новый сервер автоматически наполняется дефолтными каналами `#welcome`, `#general-chat` и голосовым залом!
   - Кнопка создания собственных текстовых (`💬`) и голосовых (`🔊`) комнат прямо внутри категорий.

3. **Интерактивные Телефонные Звонки (Direct Call)**:
   - Укажите номер телефона при создании профиля (или в любой момент в настройках пользователя) для привязки виртуальной линии!
   - Перейдите во вкладку **Direct Dial (Phone Calls)**, введите номер телефона вашего друга на клавиатуре-тачпаде или выберите контакт из списка «Доступных контактов» и нажмите **Dial**.
   - Адресату мгновенно придет анимированное окно уведомления о входящем звонке **Incoming Call** с возможностью принять (**Accept**) или отклонить (**Decline**) вызов.
   - При принятии звонка оба пользователя автоматически перенаправляются в защищенный стриминговый мост **RTC Callroom** с трансляцией аватаров и симуляцией веб-камер.

4. **Голосовые комнаты с Анализатором Микрофона**:
   - Присоединяйтесь к голосовым каналам в один клик.
   - Если у вас разрешен доступ к микрофону, приложение использует **Web Audio API Context**, чтобы замерять реальную громкость вашего голоса. Когда вы говорите перед экраном, ободок вокруг вашей аватарки начинает **светиться ярко-зеленым цветом** в реальном времени для всех пользователей на сервере!
   - Панель микрофона отображает интерактивный эквалайзер децибел (dB Analyzer).
   - Возможность быстро включить режим заглушения микрофона (**Mute**) или наушников (**Deafen**) внизу панели.

5. **Бесплатный разблокированный Discord Nitro 💎**:
   - Дополнительные анимированные неоновые ободки вокруг аватарок.
   - Доступ к фирменной панели **Nitro Stickers**: отправляйте анимированные стикеры Wumpus, Clyde, Pepe, Mario и др. прямо в текстовые чаты в оригинальном качестве.

## 🛠️ Как запустить проект локально (Инструкция для запуска)

Чтобы запустить этот сервер и клиент у себя на компьютере или сервере (после того, как вы скачали архив или выгрузили код на GitHub):

### 1. Требования
У вас должен быть установлен **Node.js** (версии 18 или выше) и пакетный менеджер **npm** (идет вместе с Node.js).
Проверить версию можно в терминале:
```bash
node -v
npm -v
```

### 2. Установка зависимостей
Перейдите в папку с проектом в вашей консоли и установите все необходимые пакеты:
```bash
npm install
```

### 3. Запуск в режиме разработки (с автообновлением)
Запустите локальный полнотекстовый сервер разработки. Он запустится на порту **3000** и свяжет React-клиент с бэкендом на Node.js:
```bash
npm run dev
```
После этого откройте в браузере: `http://localhost:3000`. 
*Совет: Откройте эту ссылку во втором окне в режиме Инкогнито, чтобы протестировать звонки и чат между двумя разными профилями локально!*

## 🚀 Сборка проекта и деплой на GitHub / Хостинг

### Сборка для продакшена (Production Build)
Чтобы собрать оптимизированный и быстрый вариант приложения для боевого сервера, выполните:
```bash
npm run build
```
Эта команда сделает две вещи:
1. Переведет React в оптимизированные статические файлы в папку `/dist`.
2. Скомпилирует TypeScript бэкенд в единую быструю Node-сборку `dist/server.cjs` с помощью инструмента `esbuild`.

### Запуск готового сервера (Start)
После успешного выполнения сборки (`npm run build`), вы можете запустить приложение одной простой командой:
```bash
npm run start
```
Сервер будет слушать на порту **3000**. Вы сможете выставить его в интернет на любом хостинге (например, VPS/VDS, Render, Heroku или Dokku), настроив перенаправление портов. Настройки состояния сохраняются в локальный файл `discord_db.json`, поэтому ваши переписки и зарегистрированные контакты не сотрутся при перезагрузках!
